Platform

Built for reliability and observability

The FastAPI backend ships with health checks, rate limits, optional Redis persistence, and a live server-sent-events stream to monitor demo usage in real time.

Backend

  • ✅ FastAPI services with auth, contact, Gmail cleanup, and demo logging routes
  • ✅ Redis-backed persistence for demo events when DEMO_STORE=redis is set
  • ✅ New /api/v1/demo/stream SSE endpoint for real-time activity feeds
  • ✅ Configurable rate limiting and spam protections on demo submissions

Frontend

  • ✅ Multi-page marketing site (Home, Solutions, Pricing, Platform)
  • ✅ Live activity module powered by SSE with polling fallback
  • ✅ Tailwind styling with reusable navigation across pages
  • ✅ Demo CTA wired to backend demo endpoints
